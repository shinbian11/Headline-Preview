# 헤드라인 미리보기 프로그램

<pre>
입력한 길이를 넘어가는 제목을 썼을 때 실제 메인화면에서 어떻게 보일지 알 수 있는 프로그램. 

예를 들어, 네이버 스포츠란의 메인 헤드라인이 너무 길면, 일정 길이 이후 제목이 ... 처리가 되는데, 
기사를 쓰는 기자 입장에서 정해진 길이 내에 가장 적절한 제목을 선택하기 위해서는
미리보기 기능이 필요하다고 생각해서 만들게 되었습니다.
</pre>
